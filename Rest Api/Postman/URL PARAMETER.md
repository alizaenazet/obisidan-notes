## Single parameter

### **`Protocol`/`Host`/`Path`?`Key`=`Value`**
contoh:
```http
https/perpustakaanKota/buku?genre=comedy
```

## Multi parameter 
**add more parameter with `&` keyword**
### **`Protocol`/`Host`/`Path`?`Key`=`Value`&`Key`=`Value`**

contoh:
```http
https/perpustakaanKota/buku?genre=comedy&size=medium
```
